Lesson 03: Strings & Formatting

Beginner⏰ 7 min

Strings are sequences of characters — letters, digits, spaces, symbols, and even emojis. They are one of the most common data types in Python, used everywhere from user interfaces to data processing. You can create a string by wrapping characters in matching quotes: single, double, or triple for multi-line strings.

The best way to embed values inside strings is with f-strings (available since Python 3.6). Prefix the string with f and insert variables directly using curly braces: f"Hello, {name}! You are {age} years old.". You can even put expressions inside the braces: f"Next year you will be {age + 1}" or call methods: f"Uppercase: {name.upper()}". F-strings are the modern, recommended approach — they are faster, cleaner, and more readable than older alternatives.

Before f-strings, developers used concatenation with the + operator ("Hello " + name) or the .format() method ("Hello {}".format(name)). Both work, but f-strings are superior in every way and are the standard in modern Python code.

Strings also offer a rich collection of built-in methods for text manipulation:

  • .upper() — converts the entire string to uppercase
  • .lower() — converts the entire string to lowercase
  • .strip() — removes leading and trailing whitespace (spaces, tabs, newlines)
  • .startswith() and .endswith() — check how the string begins or ends
  • .count() — counts how many times a substring appears

The built-in len() function returns the number of characters in a string, including spaces and punctuation. Remember that strings are immutable — methods like .upper() do not modify the original string; they return a new one. You must assign the result back to a variable if you want to keep the change.

Common mistake: Trying to modify a character directly with text[0] = "H" will raise a TypeError. Strings do not support item assignment. Instead, create a new string using slicing or .replace().

name = "JometCode"
lang = "Python"
# f-strings (Python 3.6+)
print(f"{name} is a free online {lang} IDE")
# Methods
print(lang.upper())
print(lang.lower())
print(len(lang))
Try it: Add .strip() before .upper() in the code to see method chaining in action: text.strip().upper(). Then try text.count("o") to count letter occurrences.
