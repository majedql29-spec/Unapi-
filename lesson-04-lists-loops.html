Lesson 04: Lists & Loops

Beginner⏰ 10 min

A list is an ordered collection of items that can hold any data type — numbers, strings, booleans, or even other lists. You create a list with square brackets: fruits = ["apple", "banana", "cherry"]. Lists are one of the most versatile and frequently used data structures in Python.

Access individual items by their index. Python uses zero-based indexing, meaning the first item is at position 0, not 1: fruits[0] returns "apple", fruits[1] returns "banana". Negative indices count from the end of the list: fruits[-1] is the last item ("cherry"), fruits[-2] is the second-to-last. You can also use slicing to get a sublist: fruits[1:3] returns items at positions 1 and 2 (the end index is exclusive).

To iterate over a list, use a for loop. Python's for loop is a "for-each" style — the loop variable takes each value in the collection automatically, one at a time:

for fruit in fruits:
    print(fruit)

This is more intuitive and less error-prone than traditional index-based loops found in languages like C or Java. If you need the index as you iterate, use enumerate(): for i, fruit in enumerate(fruits):.

The built-in range(n) function generates a sequence of numbers from 0 to n-1. You will often see it in combination with for loops: for i in range(5): print(i) prints 0 through 4. You can also specify a start and step: range(2, 10, 2) generates 2, 4, 6, 8.

Key list operations:

  • .append(item) — adds an item to the end
  • .insert(index, item) — inserts at a specific position
  • .remove(item) — removes the first occurrence of a value
  • .pop(index) — removes and returns an item at a given index
  • .sort() — sorts the list in place
  • len(list) — returns the number of elements

Lists are mutable, which means you can change, add, and remove elements after creation. This is different from tuples, which are fixed once created. Understanding when to use lists versus tuples is an important skill in Python programming.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
print()
# Range loop
for i in range(5):
    print(i, end=" ")
Try it: After the loop, add fruits.append("date") and then loop again to see how lists grow dynamically. Then try print(len(fruits)) to see the updated count.
