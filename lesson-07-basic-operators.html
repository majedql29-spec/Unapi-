Lesson 07: Basic Operators

Beginner⏰ 7 min

Python supports a full set of mathematical and logical operators for working with numbers, making it an excellent language for everything from simple arithmetic to complex scientific computing. Python follows the standard order of operations (PEMDAS: Parentheses, Exponents, Multiplication/Division, Addition/Subtraction).

Arithmetic operators:

  • + addition and string concatenation
  • - subtraction
  • * multiplication and string repetition ("Hi" * 3 gives "HiHiHi")
  • / division — always returns a float, even when dividing two integers (6 / 3 returns 2.0)
  • // floor division — divides and rounds down to the nearest integer
  • % modulus — returns the remainder of division, useful for checking even/odd numbers or cycling through ranges
  • ** exponentiation — raises a number to a power (2 ** 3 returns 8)

Comparison operators compare two values and return a boolean result:

  • == — equal to. Note the double equals — a single = is for assignment
  • != — not equal to
  • > — greater than
  • < — less than
  • >= — greater than or equal
  • <= — less than or equal

Logical operators combine multiple boolean expressions:

  • and — true only if both sides are true
  • or — true if at least one side is true
  • not — inverts the boolean value

Python also supports augmented assignment operators that combine an operation with assignment: +=, -=, *=, /=, etc. For example, x += 5 is shorthand for x = x + 5. These operators are common in loops and accumulators.

Important detail about //: Floor division always rounds down to the nearest integer. For positive numbers (10 // 3 = 3) this matches integer division. But for negative numbers, it rounds away from zero: -10 // 3 gives -4, not -3. The % operator follows the same sign convention, so -10 % 3 gives 2. This behavior is mathematically consistent but can surprise newcomers.

Quick tip: Use % 2 == 0 to check if a number is even, and % 2 == 1 for odd. The modulus operator is also useful for constraining values to a range, like wrapping around a circular buffer or creating repeating patterns.

a = 10
b = 3
print("Addition:", a + b)
print("Subtraction:", a - b)
print("Multiplication:", a * b)
print("Division:", a / b)
print("Floor Div:", a // b)
print("Modulus:", a % b)
print("Power:", a ** b)
print("Is a > b?", a > b)
print("Is a == b?", a == b)
Try it: Add a line using % to check if a is even: print("a is even?", a % 2 == 0). Then experiment with += by adding a += 5 and printing a again.
